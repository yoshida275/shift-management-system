# Shift Management Automation

## 概要
アルバイト先のシフト提出業務を効率化するために、
Google Apps Script（GAS）とJavaScriptを用いて作成したシステムです。

## 開発背景
以前は紙でシフトを提出しており、
毎回提出遅れが発生していました。

そのため、Googleスプレッドシートを利用した
オンライン提出方式へ変更しました。

## 機能
- シフト提出管理
- 未提出者判定
- LINE通知
- Google Sheets連携

## 使用技術
- Google Apps Script
- JavaScript
- Google Sheets
- LINE Messaging API
